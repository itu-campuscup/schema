# schema

This repository contains the shared Convex schema for the Judge IT project.
The reason is that the schema might be needed by other future projects, and it is easier to maintain it in a separate repository than to duplicate it across multiple repositories.

> Success without duplication is merely future failure in disguise\
> Randy Gage

Currently consumed by the following projects:

- [judge-it](https://github.com/itu-campuscup/judge-it)
